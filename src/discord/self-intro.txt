主にマイクラサーバーの管理をするbotです
gRPCというプロトコルで通信してます
ec2インスタンス上で動いてるよ

★コマンド
!conoha server -> サーバーの状態を確認するよ
!conoha start -> サーバーを立ち上げるよ
!conoha stop -> サーバーをシャットダウンするよ
!conoha reboot -> サーバーを再起動するよ
!vote (タイトル) (選択肢1) (選択肢2) ... -> 投票を作成するよ(隠しコマンドでリマインドしにいくよ)
